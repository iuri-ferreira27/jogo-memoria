# 🃏 Jogo da Memória

Jogo da memória clássico com emojis, cronômetro, contador de movimentos e tela de vitória animada.

## 🎮 Funcionalidades

- Grade 4×4 com 8 pares de emojis
- Embaralhamento aleatório a cada novo jogo
- Contador de movimentos
- Cronômetro em tempo real
- Animação de virada de carta (3D flip)
- Animação de erro (shake)
- Tela de vitória com seu resultado
- Design dark mode

- HTML5

## 🚀 Como executar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/iuri-ferreira27/jogo-da-memoria.git
   ```

2. Entre na pasta:
   ```bash
   cd jogo-da-memoria
   ```

3. Abra o `index.html` no navegador. Pronto!

## 💡 Conceitos aprendidos

- Algoritmo de embaralhamento (Fisher-Yates)
- Animações CSS 3D com `transform: rotateY()`
- Lógica de comparação de pares
- `setInterval` para o cronômetro
- Manipulação de classes CSS dinamicamente
